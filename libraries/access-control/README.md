# Role-Based Access Control

## Core Design

The core design of Cardinal's RBAC layer is based on two concepts: Roles and
Capabilities. These two concepts and their parts are documented on the public
Help Site page for Cardinal Media Server.

[https://help.cardinalapps.io/guides/cardinal-media-server/role-based-access-control](https://help.cardinalapps.io/guides/cardinal-media-server/role-based-access-control)

## Developer Docs

Documentation on how to work with Cardinal's RBAC is available in the general
monorepo docs. It goes beyond just this core package, and includes details on
how to use it in the UI and in the servers.
