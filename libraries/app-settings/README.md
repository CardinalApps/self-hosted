# App Settings

App Settings (aka "user preferences", aka "allowing the user to enable the dark
theme across all their apps in one fell swoop") are all of the settings that a
user can change about any of their apps. Things like the language, theme, and
start page.

Most of these settings are exposed through the `<SettingsPanel />` in the React
apps, but there are other ways to work with them too. This package provides
utils for doing that, and exports the master list of all settings.
